Create a new Vite React TypeScript project in the current directory:

1. Run `npm create vite@latest . -- --template react-ts` (use `.` for current dir)
2. Install dependencies:
   - `npm install`
   - `npm install tailwindcss @tailwindcss/vite`
   - `npm install react-dropzone @headlessui/react`

3. Configure vite.config.ts for Tailwind v4:
```typescript
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [
    react(),
    tailwindcss(),
  ],
})
```

4. Replace src/index.css content with Tailwind v4 import:
```css
@import "tailwindcss";
```

5. Create minimal src/App.tsx placeholder:
```tsx
function App() {

        Virtual Clothing Model Generator
      </h1>
    </div>
  )
}

export default App
```
